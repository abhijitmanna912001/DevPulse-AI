# DevPulse AI

DevPulse AI is an AI-powered Developer Wellness Coach for software engineers, students, competitive programmers, and remote developers. It helps users understand their health, recovery, and burnout risk by analyzing sleep, activity, mood, and work-pattern data.

Rather than only displaying raw metrics, DevPulse AI explains trends in plain language, answers wellness questions about the user's own data, and provides practical recommendations for healthier developer routines.

## Planned Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **Backend:** Node.js, Express, TypeScript
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Authentication:** JWT
- **AI:** OpenAI API
- **Deployment:** Vercel for frontend, Render or Railway for backend

## MVP Focus

- User authentication
- Manual wellness data entry
- Developer-focused wellness dashboard
- Burnout risk scoring
- AI-generated trend summaries
- AI coach chat for questions about user data
- Personalized recovery and habit recommendations
